About This Project

The House Price Predictor is a machine learning-powered web application that estimates the market value of a house based on several features provided by the user.

🎯 Objective

To offer an easy-to-use platform where anyone can input key house attributes and receive an instant predicted price, powered by a trained machine learning model.

🛠 Features

  • Responsive design using HTML, CSS, and JavaScript
  • Prediction form with real-time popup result
  • Backend developed using Flask (Python)
  • Prediction using a trained XGBoost model
  • Popup card with close (×) button to display result

📊 User Input Fields

  • Living Area
  • Grade of the House
  • Latitude & Longitude
  • Renovated Living Area
  • Area (excluding basement)
  • Waterfront presence
  • Built Year
  • Postal Code
  • Number of Views

⚙️ Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Flask (Python)
  • ML Model: XGBoost Regressor
  • Model Integration: joblib, pandas

🚀 Future Scope

  • Add interactive maps (Google Maps API)
  • User account and login functionality
  • Save and export prediction history
  • Model improvements and feature expansion
